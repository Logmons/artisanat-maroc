# 🇲🇦 Artisanat Maroc

Plateforme web développée avec **Django** pour promouvoir et vendre des produits d'artisanat marocain en ligne.

Ce projet permet de présenter différents produits artisanaux, gérer les images, et offrir une interface simple pour les utilisateurs.

---

## 🚀 Fonctionnalités

- Affichage des produits artisanaux
- Gestion des images des produits
- Interface web avec Django
- Gestion des fichiers statiques et media
- Structure modulaire avec Django Apps

---

## 🛠️ Technologies utilisées

- Python
- Django
- HTML / CSS
- SQLite
- Bootstrap (si utilisé)

---

## 📂 Structure du projet

```
artisanat_project/
│
├── artisanat_app/        # Application principale
├── artisanat_project/    # Configuration Django
├── static/               # Fichiers CSS / JS / images
├── media/                # Images uploadées
└── manage.py             # Commande principale Django
```

## ⚙️ Installation

1️⃣ Cloner le repository

```bash
git clone https://github.com/Logmons/artisanat-maroc.git
```

2️⃣ Aller dans le dossier du projet

```bash
cd artisanat-maroc
```

3️⃣ Créer un environnement virtuel

```bash
python -m venv venv
```

4️⃣ Activer l'environnement

Windows :

```bash
venv\Scripts\activate
```

Linux / Mac :

```bash
source venv/bin/activate
```

5️⃣ Installer les dépendances

```bash
pip install -r requirements.txt
```

---

## ▶️ Lancer le projet

Appliquer les migrations :

```bash
python manage.py migrate
```

Lancer le serveur :

```bash
python manage.py runserver
```

Ouvrir dans le navigateur :

```
http://127.0.0.1:8000
```
